企画の方針

今回の企画16本における共通の設計思想

Design Principle 「岩館さんから引き出す」構成を基本とする。岩館さんが既にYouTubeや日常で語っている内容を、ショート向けに「問い」の形で引き出す設計。台本ではなく会話の流れで撮れるものを優先。
  • フックは必ず疑問形で岩館さんへの「質問」として設計 → カメラに向かって話すより自然な絵が撮れる
  • AIネタを4本組み込み。岩館さんが話したがっているテーマを早めに消化する
  • 数字・実例・本音が入るとバズりやすい。「〇〇な院長」「〇〇フレーズ」などの具体性を意識
  • まず岩館さんにこの一覧を見せてOK/NG/修正をもらい、採用企画から台本作成へ進む

① 社長突撃系（5本）

岩館さんの1日・判断・本音にカメラが密着。経営者の頭の中を覗く番組型。

01-01 岩館さんが毎朝チェックする3つの数字 A / B / C
フック
「社長、朝一番に何を見てるんですか？」
内容
売上・予約数・キャンセル率など経営者が毎朝チェックするKPIを自然な会話で引き出す。「この数字が動いたら即アクション」まで話してもらうと実用性が出る。
撮れ高
スマホ画面を映しながら話せると映えやすい。スプレッドシートやダッシュボードが映ると説得力UP。
01-02 クリニックコンサルを断った瞬間の話 A / C
フック
「今まで断った案件って、どんな時ですか？」
内容
受けない案件の共通点を本音で語ってもらう。「経営者の姿勢」「数字の見方」など。視聴者が「自分はどっちか」と自己診断できる構成にする。
撮れ高
「断った理由TOP3」形式にするとテロップが作りやすく、切り抜きにも使える。
01-03 岩館さんが院長に最初に聞く質問 A / B
フック
「初回面談で必ず聞くことって何ですか？」
内容
「集患の悩みより先に聞くこと」「この質問で経営の深さがわかる」という切り口。コンサル目線のフレームワークを自然に伝える。
撮れ高
質問を「3つ」に絞ると構成しやすい。「なぜそれを聞くのか」の理由まで話してもらうと深みが出る。
01-04 AIを使い始めて、業務で一番変わったこと ✦ AI A / B / D
フック
「岩館さん、最近AIどう使ってますか？」
内容
Claude・ChatGPTなど実際に使っているツールと具体的なユースケースを話してもらう。「これだけで月〇時間削減」など数字が出ると刺さる。一般視聴者（D層）にも届くテーマ。
撮れ高
実際の画面を見せながら話すと視覚的にわかりやすい。「使う前・使った後」の変化を対比させる。
01-05 月商1000万のクリニックと3000万の違い A / C
フック
「規模が倍になる院長って、何が違うんですか？」
内容
数字の差ではなく「思考・行動・習慣」の違いを語ってもらう。「自分はどっちか」と視聴者が考えるフックを作る。
撮れ高
「伸びる院長がやっていること3つ」形式で整理するとテロップ映えする。

② 時事ニュース解説（5本）

世の中のニュースを「クリニック経営に何が変わるか」に翻訳する解説型。

02-01 ChatGPT・Claude、クリニック経営にどう使う？ ✦ AI A / B / D
フック
「院長先生、AIって実際使えてますか？」
内容
問診票・カルテ要約・SNS投稿・スタッフ教育での活用法を解説。「今すぐできる1つのこと」で締めると行動喚起になる。
撮れ高
実際の入力→出力画面を見せながら解説すると具体性が出る。「使ってない院長が損してること」という切り口も有効。
02-02 AI問診が普及したらクリニック経営はどう変わる ✦ AI A / B
フック
「AI問診導入のクリニック、もう増えてますよね？」
内容
導入した院長・していない院長の差を解説。「先行者メリットを今取るか、後で追いかけるか」という構図にすると経営判断の話になる。
撮れ高
具体的なAI問診ツール名を出せるとリアリティが増す。「導入コストvs削減できる工数」を数字で示すと保存率UP。
02-03 診療報酬改定、今回の本当のインパクト A
フック
「改定されましたけど、ぶっちゃけどのくらい影響ありますか？」
内容
数字の変化より「経営行動として何を変えるべきか」に翻訳。「対応できていないクリニックがやりがちなミス」を入れると保存率UP。
撮れ高
タイムリー性が命。改定直後に撮影・投稿できると再生数が伸びやすい。
02-04 SNS炎上したクリニック、何がまずかったのか A / B / C
フック
「最近また炎上してましたよね、あれ何がダメだったんですか？」
内容
事例ベースで「やってはいけない表現・対応」を解説。特定はせず一般化。自院の運用ルール見直しのきっかけになる内容。
撮れ高
「やりがちなNG表現TOP3」形式にすると切り抜きにもなる。炎上事例の直後に投稿するとタイムリー。
02-05 美容クリニック増加で競争激化、生き残る条件 A / C
フック
「クリニック数が増え続けてますけど、これどう見てますか？」
内容
市場飽和の現状と「指名想起を取れるクリニック」の条件を語る。「価格競争に入った瞬間に負け」という岩館さんの持論を引き出す。
撮れ高
「生き残るクリニックの共通点3つ」で整理すると視聴完了率が上がる。

③ 経営あるある（6本）

「言いづらいけどみんな抱えてる問題」を言語化。共感→笑い→学びの流れ。

03-01 伸びないクリニックの院長が言う3フレーズ A / B / C
フック
「あ〜これ言ってる院長やばいな…ってやつあります？」
内容
「うちは口コミで来る」「SNSは若い子がやること」「いい医療をすれば広まる」などを岩館さん目線でリアルに解説。視聴者の自己診断を促す。
撮れ高
3フレーズを順番に出していくテロップ構成が映える。「言いたいけど言えなかった本音」を出してもらうと視聴者の共感を得やすい。
03-02 Google口コミ1件に3時間かけた院長の話 A / B
フック
「口コミ対応、やりすぎてる院長っていますよね？」
内容
「時間対効果が最悪な行動TOP3」に口コミ返信を入れて共感→学びの流れを作る。「正しい口コミ対応の時間配分」まで話せると実用的。
撮れ高
岩館さんが実際に見た事例を話してもらうとリアリティが増す。「深夜に口コミ見てる院長の話」は多くの院長が共感する。
03-03 「集患に困ってる」と言いながら広告費ゼロの院長 A / C
フック
「集患の悩み相談で、一番多いパターンって何ですか？」
内容
「広告は怖い」「口コミだけで来るはず」という思い込みを数字で解体。投資対効果を具体的に示すと行動喚起になる。
撮れ高
「広告費0円 vs 月30万円の院長、1年後の差」のような対比構成にすると伝わりやすい。
03-04 スタッフが突然辞める前に必ず起きる3つのこと A / B / C
フック
「退職の前兆って、実はわかりやすいんですよね」
内容
「シフト希望が減る」「ランチを一人で食べ始める」など具体的なサイン。「気づいた時の正しい対処法」まで話せると完結する。
撮れ高
「退職2週間前の院長あるある」フォーマットにすると③あるあるらしさが出る。保存・共有されやすいテーマ。
03-05 業者のDMで良さそうに見えるやつの見分け方 A / B
フック
「院長宛てのDMって毎日来ますよね、どう見分けてます？」
内容
「実績が感想文だけ」「成功事例しか書いてない」「返信したら即電話」など具体的な見分け方を列挙。岩館さんが実際に断った事例を交えるとリアル感UP。
撮れ高
実際に届いたDMを（特定できない形で）見せながら解説できると映える。
03-06 AIに仕事を奪われる医師と、使いこなす医師の差 ✦ AI A / B / C / D
フック
「AIが普及したら、医師の仕事ってどうなると思いますか？」
内容
AI代替されやすい業務 vs 人にしかできない価値を整理。「使いこなす医師の共通点」を具体的に話してもらう。D層（一般視聴者）にも届くテーマで拡散に強い。
撮れ高
「AIに負ける医師の3つの特徴」「AIを武器にする医師がやっていること」の対比テロップが映える。

優先度・撮影順の目安

岩館さんの確認後、採用企画から以下の順で撮影・台本作成に進む。
